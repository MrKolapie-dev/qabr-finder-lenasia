🛠️ Technical Setup

1. Web Hosting (GitHub Pages)





Log in to your GitHub account (zahidkola@mrkolapie.com).



Create a public repository named qabr-finder-lenasia.



Upload index.html, LICENSE, and this README.md to the root directory.



Go to Settings > Pages and enable deployment from the main branch.

2. Legal Folder Setup





Inside your repository, create a folder named legal.



Upload popia.html, terms.html, and safety.html into the /legal folder.

3. Email Automation (Formspree)





Go to Formspree.io and "Get Started" using your Google account.



Create a new form project named "Qabr Finder".



Copy your unique endpoint URL (e.g., https://formspree.io/f/xyzk...).



Ensure the action URL in your index.html matches this endpoint.

4. Play Store Deployment





Use Bubblewrap CLI or PWA2APK to wrap your GitHub Pages URL (https://mrkolapie-dev.github.io/qabr-finder-lenasia/).



Upload the generated .aab file to your Google Play Console.
